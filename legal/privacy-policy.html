Política de Privacidade

Última atualização: 18 de maio de 2026

Esta Política de Privacidade descreve como o Clube do Achado ("nós", "nosso" ou "aplicativo") coleta, usa e protege as informações obtidas por meio de nossa integração com as APIs do TikTok e do Instagram, no contexto de publicação de conteúdo de marketing de afiliados.

Resumo: Coletamos apenas os dados necessários para publicar vídeos de recomendação de produtos na conta @clubedoachadoia do TikTok. Não vendemos dados pessoais. Não coletamos dados de usuários terceiros.

1. Quem somos

O Clube do Achado é um projeto de marketing de afiliados operado por Rogerio Matias, com sede em Água Boa, Mato Grosso, Brasil. Publicamos recomendações de produtos nas redes sociais, gerando comissão por meio de links de afiliado do Mercado Livre e AliExpress.

Contato: mathyass1982@gmail.com

2. Dados coletados via API do TikTok

Nossa integração com a TikTok Content Posting API coleta e processa os seguintes dados relacionados à conta TikTok do operador (@clubedoachadoia):

  • open_id — identificador único da conta TikTok (necessário para publicação)
  • display_name — nome de exibição da conta
  • access_token / refresh_token — tokens de autorização OAuth 2.0 (armazenados localmente com segurança)
  • Metadados de publicação — ID do vídeo publicado, timestamp, status da publicação

Não coletamos dados pessoais de seguidores, espectadores ou outros usuários do TikTok. Nosso aplicativo acessa apenas a conta do próprio operador mediante autorização explícita via OAuth 2.0.

3. Dados coletados via API do Instagram (Meta)

Para o sistema de resposta automática de comentários (Motor 05), coletamos:

  • Nome de usuário (@username) — de comentários públicos em nossos posts
  • Instagram Scoped User ID (IGSID) — necessário para envio de mensagens diretas
  • Texto dos comentários — para identificar intenção de compra
  • ID da mídia — para associar o comentário ao produto correto

Esses dados são usados exclusivamente para enviar, de forma automatizada, o link de afiliado do produto via mensagem direta (DM) ao usuário que demonstrou interesse. Não são compartilhados com terceiros.

4. Finalidade do uso dos dados

  • Publicação automática de vídeos de recomendação de produtos no TikTok
  • Resposta automatizada a comentários com intenção de compra no Instagram
  • Envio de links de afiliado via DM para usuários que solicitaram
  • Remarketing discreto (reengajamento de usuários que demonstraram interesse anterior)
  • Análise de desempenho interno (sem compartilhamento externo)

5. Armazenamento e segurança

Todos os dados coletados são armazenados localmente em servidores controlados pelo operador, protegidos por controle de acesso. Tokens de autenticação são armazenados em arquivo .env com acesso restrito ao sistema.

Não utilizamos serviços de nuvem de terceiros para armazenar dados pessoais de usuários (exceto Cloudinary para imagens de produtos, que não contém dados pessoais identificáveis).

Os dados de comentários e DMs são mantidos por até 90 dias para fins de deduplicação e controle de remarketing, após os quais podem ser excluídos.

6. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros. Os dados são usados exclusivamente pelo Clube do Achado para as finalidades descritas nesta política.

Integrações de terceiros utilizadas:

  • TikTok API — publicação de vídeos (Política TikTok)
  • Meta Graph API — Instagram DMs e comentários (Política Meta)
  • Mercado Livre API — dados de produtos e links de afiliado
  • AliExpress Affiliate API — dados de produtos e links de afiliado

7. Direitos dos usuários (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), os usuários têm direito a:

  • Saber quais dados seus coletamos
  • Solicitar a exclusão de seus dados
  • Revogar o consentimento para recebimento de mensagens automatizadas
  • Obter uma cópia dos dados que mantemos sobre você

Para exercer esses direitos, entre em contato: mathyass1982@gmail.com

8. Cookies e rastreamento

Este site estático não utiliza cookies, scripts de rastreamento ou tecnologias de analytics. O aplicativo em si não opera como um site visitado por usuários finais.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. A data de "última atualização" no topo desta página indica quando a versão mais recente foi publicada. Uso continuado do sistema após alterações implica aceitação da nova política.
